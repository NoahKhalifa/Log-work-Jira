# reports/

Mỗi file = một ngày làm việc. Tên file: `YYYY-MM-DD.md`.
Trong cùng một ngày, mỗi phiên là một block `## Phiên HH:MM` (append, không ghi đè).

Cấu trúc một block phiên: xem [_TEMPLATE.md](_TEMPLATE.md).

Quy trình ghi report bắt buộc khi nào: xem mục **6. Quy trình báo cáo** trong [../CLAUDE.md](../CLAUDE.md).
